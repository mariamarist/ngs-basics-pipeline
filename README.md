# NGS Basics Pipeline
Proyecto introductorio de bioinformática enfocado en el análisis básico de datos NGS (FASTQ)

# Objetivo
Aprender los primeros pasos del análisis de datos de secuenciación masiva

# Contenidos
- Conteo de lecturas FASTQ
- Control de calidad básico
- Visualización de métricas
- Notebook explicativo

# Herramientas
- Python
- R
- GitHub
